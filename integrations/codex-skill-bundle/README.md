# Write-Claw Codex Skill Bundle

This package is for Codex-style reuse inside repositories.

## Included

- `AGENTS.md`
- `.codex/skills/write-claw-runtime/SKILL.md`
- reusable templates

## Use

Copy this package into a repository where you want Codex to follow a Write-Claw style workflow.

## MCP

Not required by default.
